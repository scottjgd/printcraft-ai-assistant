## [1.0.0] — 2025-07-03

### Added
- Initial release
- Floating AI chat widget powered by OpenAI GPT-4o Mini
- 20+ pre-loaded Q&A pairs specific to Print Craft Creations
  (products, ordering, shipping, pricing, returns, contact, etc.)
- Customer feedback system (👍 / 👎 on every AI response)
- Learning system — flagged responses queue for admin review
- Human escalation — email notification when AI confidence is low
- Admin Dashboard with stats (sessions, messages, confidence, escalations)
- Conversations viewer — full chat history per session
- Knowledge Base editor — add, edit, approve, delete Q&A entries
- Escalation queue — track open/in-progress/resolved escalations
- Settings page — API key, model, bot name, greeting, colors, support email
- GitHub Updater — auto-update from GitHub Releases via WordPress dashboard
- Fully responsive chat widget (mobile & desktop)
- Typing indicator animation
- Session persistence via sessionStorage
